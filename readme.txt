A simple timer for wordpress, inspired by XKCD

==Description==

This is a small counter widget that simply shows how long it has bee since the counter was last reset. anyone is able to reset the counter using the button in the widget. This is my first wordpress widget an was used mainly as a way to teach myself how to write wordpress plugins.
